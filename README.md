# Cloud Campus
============

Cloud Campus is multi functional web application for university / college to manage campus back office data.

Ingredients
+ PHP Framework
	Cakephp 2.X (latest)
	Usermanagement Plugin (Branch cloud_campus)
	DebugKit (latest)
	CKeditor (latest)
Twitter Bootstrap U.I
	twitter bootstrap (latest)
+ Responsive Theming
	BootsWatch Themes

Infrastructure
+ Cloud Hosting
	PagodaBox
